# 📚 Month_16_Log_Data_Analysis: ระบบสั่งการและรับข้อมูลดีบักผ่าน Serial Console (CLI Menu Developer)
> การสร้างระบบติดต่อสื่อสารกับวิศวกรผ่านพอร์ต Serial (Command Line Interface) ภายในเฟิร์มแวร์ เพื่อให้อ่านค่า สแกนระบบ หรือสั่งเปลี่ยนพารามิเตอร์ VCU ทางพอร์ต USB ได้สะดวก

---
